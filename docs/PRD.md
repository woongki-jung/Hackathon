# 프로젝트 정의
본 프로젝트는 지정된 메일 주소로 수신된 이메일 내용을 분석하여 제목 및 본문의 각종 업무 관련 용어에 대한 해설을 제공하는 업무 지원 도구 개발 프로젝트이다.

## 프로젝트 진행 배경
구성원이 다양한 아이템을 담당하는 과정에서 업무수행을 위한 도메인 지식 확보가 되는 동안 업무 효율 저하 발생
도메인 역량을 강화하여 구성원간 눈높이 차이를 줄이고 바이브코딩을 활용한 생산 역량 향상 필요

## 프로젝트 목적
업무 수행 과정에서 다양한 용어에 대한 이해도를 높여 이용자의 업무 도메인 역량을 향상시키고 소통을 매끄럽게 하는 데 도움을 준다.
웹 서비스로 구동하여 접근성을 높인다.

## 실행 환경 / 주요 기술스택
- es2016, css3를 지원하는 웹/모바일 브라우저
- nextjs 15 버전
- API 기반 AI 호출 시 gemini api를 사용
- vercel 무료 인프라 사용 
- github Actions를 활용하여 di/ci 구성

## 주요 기능 요구사항
### Frontend 기능
웹 페이지로 사용자가 접속하여 사용하는 기능
#### 로그인 기능
 - 수퍼관리자 계정 및 해당 관리자가 등록한 계정만 로그인 가능
 - 수퍼관리자 계정은 별도 지정된 계정으로 처리. CLAUDE.local.md 상 초기 관리자 계정 정보 사용
 - 아이디/비밀번호는 일반적인 보안 정책을 따름
#### 환경설정 기능
- 웹훅 주소를 생성하여 분석이 필요한 정보가 발생하는 서비스에서 호출할 수 있는 기능
- 웹훅 경로별로 코드와 설명을 부여하여, 입력 데이터 분석 시 설명에 맞는 분석 방법을 사용하고 각 웹훅 별로 분류하여 확인할 수 있도록 함
#### 서비스 상태 조회기능
- 백엔드 서비스의 마지막 실행 시점과 실행 상태를 확인한다
#### 업무지원 기능
- 분석된 업무 요약사항과 우선순위, 용어해설을 파악할 수 있는 페이지
- 최신 갱신된 데이터를 우선 노출하고, 이전 이력은 목록 형태로 열어 볼 수 있도록 제공
#### 3. 용어사전 뷰어
- 백엔드에서 생성된 해설집 데이터를 조회할 수 있는 구글 검색창과 같은 입력 및 검색 폼을 제공하고, 추가로 최근 발견 빈도가 높은 용어 목록을 바로가기 형태로 확인할 수 있도록 제공한다.
### backend 기능
서비스가 실행되는 동안 백그라운드에서 상시 실행해야 하는 기능 정의. 한시간마다 지정 작업을 수행하고 데이터를 업데이트한다.
#### 1. 업무정보 수신
- 설정된 웹훅 경로로 요청이 발생하는 경우 요청 내용을 확인하여 서버 내 분석 대상 폴더에 입력 내용을 텍스트로 임시 저장한다.
#### 2. 내용 분석
분석 대상 폴더에 생성된 파일을 로드하고 claude code를 활용해 다음 동작을 수행한다.
- 메일 내용의 핵심 내용을 요약하고 후속 작업 제안을 생성한다.
- EMR 시스템 관련 용어 또는 비즈니스 용어/약어를 분류한다. 분류된 용어 별로 해설을 추출하고, 용어 - 해설 쌍이 나열된 해설집 데이터에 추가한다. 해설집 데이터는 지정 폴더에 <용어.md>와 같은 파일 형식으로 저장한다.
