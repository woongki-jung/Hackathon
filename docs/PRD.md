# 프로젝트 정의
본 프로젝트는 지정된 메일 주소로 수신된 이메일 내용을 분석하여 제목 및 본문의 각종 업무 관련 용어에 대한 해설을 제공하는 업무 지원 도구 개발 프로젝트이다.

## 프로젝트 목적
업무 수행 과정에서 다양한 용어에 대한 이해도를 높여 이용자의 업무 도메인 역량을 향상시키고 소통을 매끄럽게 하는 데 도움을 준다.

## 실행 환경 / 주요 기술스택
- windows 11 이상
- .NET 10

## 주요 기능 요구사항
### 윈도우 트레이 기능
- 실행 중 항상 윈도우 트레이 아이콘을 표시하며, 트레이 아이콘을 클릭하여 앱의 UI를 실행한다.
### 환경설정 기능
- 메일을 수신할 계정 정보(아이디, 비밀번호, 서버 주소, 포트 등)를 설정한다.
### 메일 수신 기능
- 설정된 메일 주소의 메일함을 주기적으로 확인하여 신규 메일의 내용을 읽고 지정된 분석 요청 폴더에 메일 내용을 텍스트로 저장한다.
### 용어사전 생성 기능
- 분석요청 폴더에 생성된 파일을 로드하여 내용을 분석하고, EMR 시스템 관련 용어 또는 비즈니스 용어/약어를 분류한다.
- 분류된 용어 별로 claude code를 활용해 해설을 추출하고, 용어 - 해설 쌍이 나열된 해설집에 추가한다.
### 용어사전 뷰어
- 적재된 용어목록을 사전 형태로 확인할 수 있는 뷰를 제공한다.
- 시작 화면에서 구글 검색창과 같은 입력 및 검색 폼을 제공하고, 추가로 최근 발견 빈도가 높은 용어 목록을 바로가기 형태로 확인할 수 있도록 제공한다.
