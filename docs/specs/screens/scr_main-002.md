# 검색 결과 목록 화면 정의

## 화면 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 코드 | MAIN-002 |
| 화면명 | 검색 결과 목록 |
| 경로/위치 | 메인 윈도우 (검색 상태) |
| 접근 권한 | 로컬 사용자 |
| 관련 정책 | POL-UI (UI-05), POL-TERM |
| 이전 화면 | MAIN-001 메인 화면 (검색어 입력), MAIN-003 용어 상세 ("목록으로" 클릭) |
| 다음 화면 | MAIN-003 용어 상세 (용어 선택), MAIN-001 메인 화면 (검색어 삭제) |

---

## 화면 목적

사용자가 입력한 검색어에 대한 용어 목록을 표시하여, 원하는 용어를 빠르게 찾고 상세 정보에 접근할 수 있도록 한다.

---

## 화면 구성 요소

### 타이틀 바 영역

MAIN-001과 동일 (앱 공통 타이틀 바).

### 검색 영역 (화면 상단)

| 컴포넌트 | 타입 | 표시 조건 | 설명 |
|----------|------|-----------|------|
| 검색 입력창 | TextBox | 항상 | 현재 검색어가 입력된 상태, 상단 고정 |
| 검색어 지우기 버튼 | Button (X) | 검색어 입력 중 | 클릭 시 검색어 삭제, MAIN-001로 복귀 |
| 결과 건수 | Label | 검색 결과가 있을 때 | "검색 결과 {n}건" |

### 검색 결과 목록 영역 (검색 영역 아래)

| 컴포넌트 | 타입 | 표시 조건 | 설명 |
|----------|------|-----------|------|
| 결과 목록 | ListView | 검색 결과가 있을 때 | 용어 항목 목록, 스크롤 가능 |
| 용어 항목 | ListItem | 각 결과별 | 하위 구성: 용어명, 카테고리 태그, 1줄 요약 |
| 용어명 | Label (굵게) | 각 항목 | 용어 원문, 검색어 하이라이트 |
| 카테고리 태그 | Badge | 각 항목 | EMR / Business / Abbreviation, 카테고리별 색상 |
| 1줄 요약 | Label | 각 항목 | 해설의 1줄 요약 (50자 이내) |
| 빈 검색 결과 메시지 | EmptyState | 검색 결과가 없을 때 | "일치하는 용어가 없습니다" |

### 상태 바 영역 (화면 하단)

MAIN-001과 동일 (앱 공통 상태 바).

## 세부 기능 정의

### 실시간 검색

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 검색 입력창의 텍스트를 변경 |
| 처리 | 300ms 디바운스 후 용어 사전에서 검색 실행 |
| 성공 결과 | 검색 결과 목록 갱신 |
| 실패 결과 | 해당 없음 (로컬 검색) |

**검색 정렬 규칙 (POL-UI UI-05):**
1. 용어 원문 정확 일치
2. 용어 원문 부분 일치
3. 해설 텍스트 포함

### 용어 선택

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 검색 결과 목록의 용어 항목을 클릭 |
| 처리 | 선택한 용어의 상세 화면(MAIN-003) 표시 |
| 성공 결과 | 용어 상세 화면으로 전환 |
| 실패 결과 | 해당 없음 |

### 검색어 삭제

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 검색어 지우기(X) 버튼을 클릭하거나 검색어를 모두 삭제 |
| 처리 | 검색 입력창 초기화, 메인 화면(MAIN-001) 상태로 복귀 |
| 성공 결과 | 빈도 높은 용어 바로가기가 다시 표시됨 |
| 실패 결과 | 해당 없음 |

## 화면 상태 정의

| 상태 | 발생 조건 | 표시 내용 |
|------|-----------|-----------|
| 검색 중 | 디바운스 대기 중 (300ms) | 기존 결과 유지 또는 로딩 인디케이터 |
| 결과 있음 | 검색 결과가 1건 이상 | 결과 건수 + 용어 항목 목록 |
| 결과 없음 | 검색 결과가 0건 | "일치하는 용어가 없습니다" 메시지 |

---

## 화면 전환 정의

| 트리거 | 이동 화면 | 전환 방식 | 전달 데이터 |
|--------|-----------|-----------|-------------|
| 용어 항목 클릭 | MAIN-003 용어 상세 | 화면 내 뷰 전환 | 용어 ID/term |
| 검색어 지우기(X) 클릭 | MAIN-001 메인 화면 | 화면 내 뷰 전환 | 없음 |
| 검색어 모두 삭제 (백스페이스) | MAIN-001 메인 화면 | 화면 내 뷰 전환 | 없음 |
| X 버튼 (타이틀 바) 클릭 | TRAY-001 트레이 아이콘 | 윈도우 숨김 | 없음 |

## 비즈니스 규칙 및 표시 조건

- 검색은 부분 일치 방식이며, 한글/영문 대소문자 구분 없이 수행한다 (POL-UI UI-05).
- 검색어 입력 시 300ms 디바운스를 적용하여 실시간 검색한다 (POL-UI UI-05).
- 검색 결과 목록에서 용어명 내 검색어 부분을 시각적으로 하이라이트 처리한다.
- 카테고리별 색상 구분을 적용한다: EMR, Business, Abbreviation (POL-UI UI-06).
- 관련 정책 문서: docs/specs/policies/policy_pol-ui.md
