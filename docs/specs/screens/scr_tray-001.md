# 트레이 아이콘 화면 정의

## 화면 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 코드 | TRAY-001 |
| 화면명 | 트레이 아이콘 |
| 경로/위치 | Windows 시스템 트레이 영역 (작업 표시줄 알림 영역) |
| 접근 권한 | 로컬 사용자 (앱 실행 시 자동 표시) |
| 관련 정책 | POL-UI (UI-01, UI-02) |
| 이전 화면 | 없음 (앱 실행 시 자동 생성) |
| 다음 화면 | MAIN-001 용어사전 뷰어 메인 (좌클릭), TRAY-002 컨텍스트 메뉴 (우클릭) |

---

## 화면 목적

앱이 백그라운드에서 실행 중임을 사용자에게 시각적으로 알리고, 메인 윈도우 접근 및 컨텍스트 메뉴 진입점을 제공한다. 또한 메일 수신 완료, 오류 발생 등 주요 이벤트를 풍선 알림으로 전달하여 사용자가 적시에 정보를 확인할 수 있도록 한다.

---

## 화면 구성 요소

### 트레이 아이콘 영역

| 컴포넌트 | 타입 | 표시 조건 | 설명 |
|----------|------|-----------|------|
| 앱 아이콘 | NotifyIcon | 앱 실행 중 항상 | 시스템 트레이에 표시되는 앱 아이콘 |
| 툴팁 텍스트 | Tooltip | 아이콘 마우스 오버 | 앱 이름 및 현재 상태 (예: "용어해설 도구 - 동작 중") |

### 풍선 알림 영역

| 컴포넌트 | 타입 | 표시 조건 | 설명 |
|----------|------|-----------|------|
| 알림 제목 | BalloonTitle | 이벤트 발생 시 | 알림 유형에 따른 제목 |
| 알림 메시지 | BalloonText | 이벤트 발생 시 | 이벤트 상세 내용 |

## 세부 기능 정의

### 트레이 아이콘 좌클릭 - 메인 윈도우 열기

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 트레이 아이콘을 좌클릭 |
| 처리 | 메인 윈도우(MAIN-001)를 표시하거나, 이미 열려있으면 포커스 |
| 성공 결과 | 메인 윈도우가 전면에 표시됨 |
| 실패 결과 | 해당 없음 |

### 트레이 아이콘 우클릭 - 컨텍스트 메뉴 표시

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 트레이 아이콘을 우클릭 |
| 처리 | 컨텍스트 메뉴(TRAY-002) 표시 |
| 성공 결과 | 메뉴가 마우스 위치 근처에 표시됨 |
| 실패 결과 | 해당 없음 |

### 풍선 알림 표시

| 항목 | 내용 |
|------|------|
| 트리거 | 시스템 이벤트 발생 (신규 용어 추가, 오류, 디스크 부족) |
| 처리 | 트레이 아이콘에서 풍선 알림 표시 (5초간) |
| 성공 결과 | 사용자에게 알림이 표시됨 |
| 실패 결과 | 해당 없음 (OS 알림 시스템 의존) |

**풍선 알림 유형:**

| 이벤트 | 알림 제목 | 알림 메시지 |
|--------|-----------|-------------|
| 용어 분석 완료 | 새 용어 추가 | "{n}건의 새로운 용어가 추가되었습니다" |
| 메일 수신 오류 연속 5회 | 연결 오류 | "메일 서버 연결에 문제가 있습니다" |
| 디스크 용량 부족 (1GB 미만) | 저장 공간 부족 | "저장 공간이 부족합니다" |

### 풍선 알림 클릭 - 메인 윈도우 열기

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 풍선 알림을 클릭 |
| 처리 | 메인 윈도우(MAIN-001) 표시 |
| 성공 결과 | 메인 윈도우가 전면에 표시됨 |
| 실패 결과 | 해당 없음 |

## 화면 상태 정의

| 상태 | 발생 조건 | 표시 내용 |
|------|-----------|-----------|
| 정상 동작 | 앱이 정상 실행 중 | 기본 앱 아이콘, 툴팁: "용어해설 도구 - 동작 중" |
| 오류 상태 | 메일 연결 실패 연속 5회 | 아이콘에 경고 표시(선택), 툴팁: "용어해설 도구 - 연결 오류" |
| 처리 중 | 메일 수신/분석 진행 중 | 툴팁: "용어해설 도구 - 메일 확인 중..." |

| 트리거 | 이동 화면 | 전환 방식 | 전달 데이터 |
|--------|-----------|-----------|-------------|
| 트레이 아이콘 좌클릭 | MAIN-001 용어사전 뷰어 메인 | 윈도우 표시/포커스 | 없음 |
| 트레이 아이콘 우클릭 | TRAY-002 컨텍스트 메뉴 | 팝업 메뉴 표시 | 없음 |
| 풍선 알림 클릭 | MAIN-001 용어사전 뷰어 메인 | 윈도우 표시/포커스 | 없음 |
| 메인 윈도우 X 버튼 (진입) | TRAY-001 트레이 아이콘 (유지) | 윈도우 숨김, 트레이 유지 | 없음 |

## 알림/피드백 정의

| 상황 | 피드백 유형 | 메시지 |
|------|-------------|--------|
| 용어 분석 완료 | 풍선 알림 (정보) | "{n}건의 새로운 용어가 추가되었습니다" |
| 메일 서버 연결 오류 | 풍선 알림 (경고) | "메일 서버 연결에 문제가 있습니다" |
| 디스크 용량 부족 | 풍선 알림 (경고) | "저장 공간이 부족합니다" |

---

## 비즈니스 규칙 및 표시 조건

- 앱 실행 시 트레이 아이콘은 반드시 표시되어야 한다 (POL-UI UI-01).
- 메인 윈도우 X 버튼 클릭 시 앱을 종료하지 않고 트레이로 최소화한다 (POL-UI UI-01).
- 앱 종료는 오직 트레이 컨텍스트 메뉴의 "종료" 항목으로만 가능하다 (POL-UI UI-01).
- 풍선 알림 표시 시간은 5초이다 (POL-UI UI-02).
- 관련 정책 문서: docs/specs/policies/policy_pol-ui.md
