# 환경설정 다이얼로그 화면 정의

## 화면 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 코드 | SET-001 |
| 화면명 | 환경설정 다이얼로그 |
| 경로/위치 | 모달 다이얼로그 (메인 윈도우 위에 표시) |
| 접근 권한 | 로컬 사용자 |
| 관련 정책 | POL-UI (UI-03), POL-AUTH, POL-MAIL, POL-DATA, POL-TERM |
| 이전 화면 | TRAY-002 컨텍스트 메뉴 ("환경설정" 클릭) |
| 다음 화면 | (다이얼로그 닫힘) 메인 윈도우로 복귀 |

---

## 화면 목적

메일 수신, 데이터 저장, 용어 분석에 필요한 설정값을 사용자가 확인하고 변경할 수 있도록 한다. Azure AD 인증은 환경변수로 관리하므로 계정 연결 상태만 표시하며, 나머지 수신/저장/분석 설정을 편집할 수 있다.

## 세부 기능 정의

### 설정 불러오기

| 항목 | 내용 |
|------|------|
| 트리거 | 다이얼로그가 열릴 때 |
| 처리 | 현재 설정값을 읽어 각 입력 필드에 표시 |
| 성공 결과 | 모든 설정 항목에 현재 값이 표시됨 |
| 실패 결과 | 설정 파일 읽기 실패 시 기본값으로 표시하고 상단에 경고 메시지 |

### 폴더 선택

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "찾아보기..." 버튼을 클릭 |
| 처리 | Windows 폴더 선택 다이얼로그(FolderBrowserDialog) 표시 |
| 성공 결과 | 선택한 폴더 경로가 분석 요청 폴더 경로 필드에 반영 |
| 실패 결과 | 사용자가 취소한 경우 기존 경로 유지 |

### 불용어 목록 편집

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "불용어 목록 편집" 버튼을 클릭 |
| 처리 | 불용어 편집 영역을 표시/숨김 토글 |
| 성공 결과 | 불용어 편집 영역이 표시되고, 현재 불용어 목록(쉼표 구분)이 표시됨 |
| 실패 결과 | 해당 없음 |

### 설정 저장

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "저장" 버튼을 클릭 |
| 처리 | 입력값 유효성 검사 후, 설정 파일에 저장하고 즉시 적용 |
| 성공 결과 | 설정 저장 완료, 다이얼로그 닫힘, 변경된 설정 즉시 반영 (앱 재시작 불필요) |
| 실패 결과 | 유효성 검사 실패 시 해당 필드에 오류 메시지 표시, 저장 실패 시 오류 안내 |

**입력 유효성 검사:**

| 필드 | 유효성 규칙 | 오류 메시지 |
|------|-------------|-------------|
| 폴링 주기 | 30 이상 3600 이하 정수 | "30~3600 사이의 값을 입력해주세요" |
| 1회 최대 조회 수 | 1 이상 100 이하 정수 | "1~100 사이의 값을 입력해주세요" |
| 분석 요청 폴더 경로 | 유효한 디렉터리 경로 | "유효한 폴더 경로를 선택해주세요" |
| 일일 최대 API 호출 횟수 | 1 이상 정수 | "1 이상의 값을 입력해주세요" |

**처리 흐름:**
1. 사용자가 "저장" 클릭
2. 모든 입력 필드 유효성 검사 실행
3. 유효성 검사 실패 시 첫 번째 오류 필드에 포커스, 오류 메시지 표시
4. 유효성 검사 성공 시 설정 파일에 저장
5. 저장 성공 시 설정을 런타임에 즉시 반영
6. 다이얼로그 닫기

### 설정 취소

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "취소" 버튼을 클릭 또는 다이얼로그 X 버튼 클릭 |
| 처리 | 변경 사항 폐기, 다이얼로그 닫기 |
| 성공 결과 | 다이얼로그 닫힘, 설정 변경 없음 |
| 실패 결과 | 해당 없음 |

## 화면 상태 정의

| 상태 | 발생 조건 | 표시 내용 |
|------|-----------|-----------|
| 정상 로드 | 다이얼로그 열기 성공 | 모든 설정 항목에 현재 값 표시 |
| 설정 파일 오류 | 설정 파일 읽기 실패 | 기본값으로 표시 + 상단 경고: "설정 파일을 읽을 수 없습니다. 기본값으로 표시합니다." |
| 유효성 오류 | 저장 시 유효성 검사 실패 | 오류 필드 강조 + 오류 메시지 표시 |
| 계정 연결 성공 | Graph API 인증 성공 상태 | 연결 상태: 녹색 "연결됨" |
| 계정 연결 실패 | Graph API 인증 실패 또는 환경변수 미설정 | 연결 상태: 빨간색 "연결 안 됨" |

| 설정 파일 읽기 실패 | 다이얼로그 내 경고 메시지 | "설정 파일을 읽을 수 없습니다. 기본값으로 표시합니다." |

## 비즈니스 규칙 및 표시 조건

- 환경설정은 별도 모달 다이얼로그로 제공한다 (POL-UI UI-03).
- Azure AD 인증 정보(AZURE_TENANT_ID, AZURE_CLIENT_ID, AZURE_CLIENT_SECRET)는 환경변수로 관리하므로, UI에서는 계정 연결 상태만 표시한다 (POL-UI UI-03, POL-AUTH AUTH-02).
- 설정 변경은 앱 재시작 없이 즉시 반영된다 (POL-UI UI-03).
- 폴링 주기는 30~3600초 범위를 벗어나면 기본값 60초를 적용한다 (POL-MAIL MAIL-01).
- 불용어 목록은 사용자가 편집 가능하도록 한다 (POL-TERM TERM-02).
- 인증 정보는 평문으로 설정 파일에 기록하지 않는다 (POL-AUTH AUTH-02).
- 관련 정책 문서: docs/specs/policies/policy_pol-ui.md, docs/specs/policies/policy_pol-auth.md, docs/specs/policies/policy_pol-mail.md, docs/specs/policies/policy_pol-data.md, docs/specs/policies/policy_pol-term.md
