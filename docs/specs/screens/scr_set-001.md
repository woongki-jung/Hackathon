## 화면 목적

관리자가 메일 서버 접속 정보(IMAP)와 분석 설정을 조회하고 수정할 수 있다. 설정 변경 후 메일 서버 연결 테스트를 수행하여 정상 동작을 확인할 수 있다. 민감 정보(비밀번호, API 키)는 환경변수(.env.local)로 관리되며, 이 화면에서는 설정 여부만 확인할 수 있다.

## 세부 기능 정의

### 환경설정 조회

| 항목 | 내용 |
|------|------|
| 트리거 | 화면 진입 시 자동 실행 |
| 처리 | 현재 저장된 설정값 조회 |
| 관련 API | CFG-001 `GET /api/config` |
| 관련 기능 | CMN-CFG-001 |
| 성공 결과 | 각 입력 필드에 현재 설정값 표시 |
| 실패 결과 | 오류 메시지 + 재시도 버튼 |

### 환경설정 저장

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "설정 저장" 버튼을 클릭 |
| 처리 | 변경된 설정값을 서버에 저장 |
| 관련 API | CFG-002 `PUT /api/config` |
| 관련 기능 | CMN-CFG-001 |
| 성공 결과 | 토스트: "설정이 저장되었습니다." + 갱신된 값으로 폼 업데이트 |
| 실패 결과 | 토스트: 오류 메시지 |

**입력 유효성 검사 (클라이언트)**:

| 필드 | 유효성 규칙 | 오류 메시지 |
|------|-------------|-------------|
| IMAP 호스트 | 빈 문자열 불가 (입력한 경우) | "IMAP 서버 주소를 입력해주세요." |
| IMAP 포트 | 1~65535 범위 정수 | "포트 번호는 1~65535 사이의 숫자여야 합니다." |
| 메일 확인 주기 | 최소 1분(60000ms), 정수 | "확인 주기는 최소 1분 이상이어야 합니다." |
| 모델명 | 빈 문자열 불가 (입력한 경우) | "모델명을 입력해주세요." |

**처리 흐름:**
1. 사용자가 설정값을 수정한다.
2. "설정 저장" 버튼을 클릭한다.
3. 클라이언트에서 유효성을 검사한다.
4. 유효성 통과 시 `PUT /api/config` API를 호출한다.
5. 호출 중 저장 버튼을 로딩 상태로 전환한다.
6. 성공 시 토스트 메시지를 표시하고 폼을 갱신된 값으로 업데이트한다.
7. 실패 시 에러 토스트를 표시한다.

### 메일 서버 연결 테스트

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "연결 테스트" 버튼을 클릭 |
| 처리 | 현재 DB 설정 + 환경변수 비밀번호로 IMAP 연결 테스트 수행 |
| 관련 API | CFG-003 `POST /api/config/test-mail` |
| 관련 기능 | MAIL-RECV-001 |
| 성공 결과 (연결 성공) | 토스트: "메일 서버 연결에 성공했습니다. (읽지 않은 메일: N건)" |
| 성공 결과 (연결 실패) | 토스트(오류): "메일 서버 연결에 실패했습니다. 설정을 확인해주세요." |
| 실패 결과 | 토스트(오류): "연결 테스트 중 오류가 발생했습니다." |

**처리 흐름:**
1. 사용자가 "연결 테스트" 버튼을 클릭한다.
2. 변경사항이 아직 저장되지 않은 경우 "먼저 설정을 저장해주세요." 안내를 표시한다.
3. `POST /api/config/test-mail` API를 호출한다.
4. 호출 중 테스트 버튼을 로딩 상태로 전환한다 (타임아웃 10초).
5. 결과에 따라 성공 또는 실패 토스트를 표시한다.

### 변경 취소

| 항목 | 내용 |
|------|------|
| 트리거 | 사용자가 "변경 취소" 버튼을 클릭 |
| 처리 | 폼의 모든 입력값을 서버에서 마지막으로 조회한 값으로 복원 |
| 관련 API | 없음 (클라이언트 상태 복원) |
| 성공 결과 | 폼이 원래 값으로 복원됨 |
| 실패 결과 | 없음 |

## 비즈니스 규칙 및 표시 조건

- 환경설정 변경은 관리자만 가능하다 (AUTH-R-014). 일반 사용자는 이 페이지에 접근할 수 없다.
- IMAP 비밀번호와 Claude API 키는 DB/API 응답에 포함하지 않는다 (AUTH-R-017, AUTH-R-018). 설정 여부만 boolean으로 표시한다 (AUTH-R-020).
- 환경변수(.env.local)는 웹 API로 변경할 수 없으므로, 비밀번호/API 키 변경 안내 문구를 표시한다 (DATA-R-020).
- 메일 확인 주기의 최소값은 1분(60000ms)이다.
- 연결 테스트 타임아웃은 10초이다.
- 비밀번호 필드는 마스킹 처리한다 (UI-R-010).
- 관련 정책 문서: `docs/specs/policies/policy_pol-auth.md`, `docs/specs/policies/policy_pol-data.md`
