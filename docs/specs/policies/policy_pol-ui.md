# UI/UX 정책 정의

## 개요
- 웹 서비스의 사용자 인터페이스, 반응형 디자인, 브라우저 호환성, 접근성에 관한 정책을 정의한다.
- PRD의 "실행 환경"(ES2016, CSS3 지원 웹/모바일 브라우저) 및 프론트엔드 기능 요구사항에 기반한다.
- Next.js 15 App Router + Tailwind CSS 기반 구현을 전제로 한다.

---

### 예외 사항
- 로그인 페이지는 네비게이션 바(메뉴)를 표시하지 않는다.
- 관리자 전용 메뉴(환경설정, 사용자 관리)는 일반 사용자에게 노출하지 않는다.

### 구현 가이드
- 스타일링: Tailwind CSS를 사용하여 유틸리티 퍼스트 방식으로 스타일링한다.
- 라우팅: Next.js 15 App Router의 파일 기반 라우팅을 사용한다.
- 서버 컴포넌트: 데이터 페칭이 필요한 페이지는 React Server Components를 우선 사용하고, 인터랙션이 필요한 부분만 Client Components로 분리한다.
- 검색: 디바운스는 클라이언트 사이드에서 처리하며, 서버 사이드에서 검색 쿼리를 수행한다.
