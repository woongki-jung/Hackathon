# 로깅 기능 정의

## 개요
- 구조화된 로그 기록, 로그 레벨 관리, 감사 로그 기능을 정의한다.
- 적용 범위: 모든 서비스 레이어, 백그라운드 작업, API 요청 처리

---

## CMN-LOG-001 로깅

### 기본 정보
| 항목 | 내용 |
|------|------|
| 기능명 | 로깅 |
| 분류 | 공통 기능 |
| 레이어 | lib/logger |
| 트리거 | 모든 기능 수행 시 |
| 관련 정책 | POL-MAIL (MAIL-R-002, MAIL-R-003, MAIL-R-006), POL-TERM (TERM-R-006, TERM-R-007) |

### 입력 / 출력

#### createLogger (로거 인스턴스 생성)

##### 입력 (Input)
| 파라미터 | 타입 | 필수 | 설명 | 유효성 규칙 |
|----------|------|------|------|-------------|
| module | string | ✅ | 모듈/기능명 | 예: "mail-recv", "term-gen" |

##### 출력 (Output)
| 항목 | 타입 | 설명 |
|------|------|------|
| logger | Logger | info(), warn(), error(), debug() 메서드를 가진 로거 |

#### 로그 기록 (logger.info / warn / error / debug)

##### 입력 (Input)
| 파라미터 | 타입 | 필수 | 설명 | 유효성 규칙 |
|----------|------|------|------|-------------|
| message | string | ✅ | 로그 메시지 | - |
| meta | Record<string, unknown> | ❌ | 추가 메타데이터 | - |

##### 출력 (Output)
| 항목 | 타입 | 설명 |
|------|------|------|
| - | void | 콘솔(stdout/stderr)에 구조화된 로그 출력 |

### 로그 형식

```
[ISO8601] [LEVEL] [MODULE] MESSAGE {meta}
```

예시:
```
[2026-03-15T10:30:00.000Z] [INFO] [mail-recv] IMAP 연결 성공 {"host":"imap.gmail.com","mailCount":5}
[2026-03-15T10:30:01.000Z] [ERROR] [term-gen] Claude API 호출 실패 {"attempt":2,"error":"timeout"}
```

### 로그 레벨

| 레벨 | 용도 | 예시 |
|------|------|------|
| ERROR | 서비스 실패, 복구 불가 오류 | API 호출 실패, DB 연결 오류 |
| WARN | 비정상이지만 서비스 가능 | 재시도 발생, 건너뜀 발생 |
| INFO | 정상 작업 완료, 주요 이벤트 | 메일 수신 완료, 분석 완료 |
| DEBUG | 상세 실행 정보 (개발용) | 쿼리 내용, 파라미터 값 |

### 구현 가이드

- **패턴**: Factory 패턴 (createLogger로 모듈별 로거 생성)
- **구현**: console.log/error 래핑 또는 pino/winston 등 경량 로거 라이브러리
- **보안**: 민감정보(비밀번호, API 키)는 로그에 포함하지 않음
- **성능**: 동기식 콘솔 출력 (better-sqlite3 동기 환경과 호환)
- **외부 의존성**: 없음 (console 사용) 또는 선택적 로깅 라이브러리

### 관련 기능
- **이 기능을 호출하는 기능**: 모든 기능
- **이 기능이 호출하는 기능**: 없음

### 테스트 시나리오

| 시나리오 | 입력 조건 | 기대 결과 |
|----------|-----------|-----------|
| INFO 로그 | logger.info("작업 완료") | 타임스탬프+레벨+모듈+메시지 출력 |
| ERROR + 메타 | logger.error("실패", {code: 500}) | 메타데이터 포함 출력 |
| 모듈별 로거 | createLogger("mail-recv") | [mail-recv] 접두사 로그 |
