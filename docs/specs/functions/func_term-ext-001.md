# 용어 추출 기능 정의

## 개요

- **기능 목적**: 메일 텍스트에서 패턴 매칭을 통해 용어 후보를 추출한다.
- **적용 범위**: 배치 분석(TERM-BATCH-001)에서 각 파일의 텍스트에 대해 호출된다.

---

## TERM-EXT-001: 용어 추출

### 기본 정보

| 항목 | 내용 |
|------|------|
| 기능명 | 용어 추출 |
| 분류 | 도메인 특화 로직 |
| 레이어 | Domain |
| 트리거 | TERM-BATCH-001에서 파일 텍스트 분석 시 |
| 관련 정책 | POL-TERM (TERM-01, TERM-02) |

### 입력 / 출력

#### 입력 (Input)

| 파라미터 | 타입 | 필수 | 설명 | 유효성 규칙 |
|----------|------|------|------|-------------|
| text | string | ✅ | 분석 대상 텍스트 (제목+본문) | 비어 있지 않은 문자열 |
| maxTerms | int | | 최대 추출 용어 수 | 기본값 50 (TERM-05) |

#### 출력 (Output)

| 항목 | 타입 | 설명 |
|------|------|------|
| candidates | TermCandidate[] | 추출된 용어 후보 목록 |

**TermCandidate 구조**

| 필드 | 타입 | 설명 |
|------|------|------|
| term | string | 용어 원문 |
| frequency | int | 텍스트 내 출현 빈도 |
| context | string | 용어 전후 200자 컨텍스트 |
| matchPattern | string | 매칭된 패턴 유형 |

#### 예외 / 오류

| 조건 | 오류 코드 | 설명 |
|------|-----------|------|
| 빈 텍스트 | ERR_TERM_EMPTY_TEXT | 분석 대상이 비어 있음 |

### 처리 흐름

1. **패턴 매칭**: 텍스트에서 다음 패턴에 해당하는 토큰을 추출한다 (TERM-02):
   - **대문자 약어**: 2글자 이상 연속 대문자 (`[A-Z]{2,}`)
   - **코드성 용어**: 영문+숫자/하이픈 조합 (`[A-Za-z]+[-]?\d+[A-Za-z\d]*`, 예: ICD-10, HL7v2)
   - **한글 전문 용어**: 사용자 등록 용어 패턴 목록과 매칭
2. **최소 길이 필터**: 2글자 미만 문자열은 제외한다 (TERM-01).
3. **빈도 집계**: 각 용어의 텍스트 내 출현 빈도를 집계한다.
4. **컨텍스트 추출**: 각 용어의 첫 번째 출현 위치 전후 200자를 컨텍스트로 저장한다 (TERM-03 프롬프트 용도).
5. **상한 적용**: maxTerms를 초과하면 빈도순 상위 항목만 유지한다 (TERM-05).
6. **결과 반환**: TermCandidate 목록을 반환한다.

### 구현 가이드

- **패턴**: 순수 함수로 구현. 정규식 패턴은 컴파일하여 재사용.
- **성능**: 대용량 텍스트(1MB)에 대해 단일 패스로 처리 가능하도록 설계. 정규식은 사전 컴파일(Compiled 옵션).
- **확장성**: 한글 전문 용어 패턴 목록은 외부 파일에서 로드하여 사용자가 편집 가능하도록 설계.

### 관련 기능

- **이 기능을 호출하는 기능**: TERM-BATCH-001
- **이 기능이 호출하는 기능**: TERM-EXT-002 (불용어 필터링)

### 테스트 시나리오

| 시나리오 | 입력 조건 | 기대 결과 |
|----------|-----------|-----------|
| 대문자 약어 추출 | "EMR 시스템과 PACS 연동" | EMR, PACS 추출 |
| 코드성 용어 | "ICD-10 코드 및 HL7v2 메시지" | ICD-10, HL7v2 추출 |
| 2글자 미만 제외 | "I, A, OK 확인" | OK만 추출 (I, A 제외) |
| 빈도 집계 | "EMR EMR EMR PACS" | EMR(3), PACS(1) |
| 상한 적용 | 70개 용어, maxTerms=50 | 빈도 상위 50개만 |
| 컨텍스트 추출 | 용어 위치 전후 200자 | context 필드에 저장 |
| 빈 텍스트 | text="" | ERR_TERM_EMPTY_TEXT |
