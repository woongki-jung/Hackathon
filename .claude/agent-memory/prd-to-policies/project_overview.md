프로젝트: 메일 수신 용어 해설 업무 지원 도구 (EMR/의료 도메인)

**Why:** 업무 수행 과정에서 다양한 용어에 대한 이해도를 높여 소통을 원활히 하기 위한 도구

**How to apply:**
- 기술 스택: Windows 11+, .NET 10, Microsoft Graph API (OAuth 2.0), Claude API
- 주요 기능: 트레이 앱, 환경설정, 메일 수신, 용어 분석/사전 생성, 용어사전 뷰어
- 2026-03-13 기준으로 정책 정의 문서(POL-AUTH, POL-MAIL, POL-DATA, POL-TERM, POL-UI) 작성 완료
- UI 프레임워크 후보: WinUI 3 또는 WPF
